   ```sh
   git clone https://github.com/SURAJVERMA-BIT/SCREENSCALE.git

   ```

   ```sh
   pip install -r requirements.txt
   ```

## Controls
- Drag the ruler with your mouse.
- Move using keys:
1. W: Move up
2. A: Move left
3. S: Move down
4. D: Move right
5. ESC: Close the ruler
